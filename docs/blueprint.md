# **App Name**: EduGate SMK

## Core Features:

- Google OAuth Login: Allow students and teachers to securely login using their Google accounts. The .env file will hold the API key for Google OAuth.
- Dashboard: Personalized dashboard displaying student schedules, grades, and important announcements.
- Course Management: Enable teachers to manage course content, assignments, and grading.
- Gradebook: Allow teachers to submit and manage grade information.
- Schedule Management: Displays each student and teacher's specific schedule and allows admin to set them
- Notifications: Sends notifications and announcements about course updates, school events, and important deadlines.

## Style Guidelines:

- Primary color: Deep sky blue (#3498DB) to convey trustworthiness and intelligence.
- Background color: Light gray (#ECF0F1) for a clean and professional look.
- Accent color: Emerald green (#2ECC71) to highlight important actions and notifications.
- Body and headline font: 'PT Sans', a humanist sans-serif for headlines or body text.
- Use simple, modern icons to represent different sections and actions.
- Clean and intuitive layout with a focus on user-friendliness and easy navigation.
- Subtle animations for transitions and feedback to enhance user engagement.
